# Classificador de tags do stackoverflow

* Este projeto tem como foco explorar os tipos de classificação multilabel e suas métricas. Durante esta análise, fez-se a classificação das tags das perguntas do stackoverflow.

Aqui é possível encontrar dois notebooks: O notebook onde foi desenvolvido todos os códigos acerca da classificação das tags e o notebook que aborda alguns pontos sobre a teoria, caso surjam dúvidas durante a leitura.

Os dados utilizados foram obtidos no curso ícone Classificação multilabel de textos: múltiplos contextos em NLP Classificação multilabel de textos: múltiplos contextos em NLP da Alura cursos de tecnologia.
